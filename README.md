# Álbum Panini Copa 2026

Gerenciador de figurinhas da Copa do Mundo 2026.

## Como usar

Acesse: **https://eliseucazarotto1989-dot.github.io/figurinhas-copa-2026/**

Marque suas figurinhas conquistadas. Os dados ficam salvos no seu próprio navegador (`localStorage`), então cada pessoa tem a coleção dela e ninguém vê a do outro.

## Instalar como app

No celular, abra o link no Chrome/Safari e use "Adicionar à tela inicial".
